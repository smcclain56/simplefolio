Victim Advocacy Hub

I worked with two other students in my class to construct a "web portal" for secure (HIPPAA/PHI) communication streams between victims of sexual assault and victim advocates. I designed a relational database solution in SQLite and implemented various front-end interfaces using HTML, CSS, and PHP for both victims and victim advocates. We used the SendBird API for sending secure messages, and deployed the application onto an Apache Server. As a team we engaged in a daily scrum log and alternated scrum master responsibilites.
